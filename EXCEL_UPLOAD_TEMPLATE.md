# Sample Excel Upload Template for Installation Sites

This file should be used as a template for uploading new installation locations.

## Required Columns (in order):
1. **Project Name** - Name of the project (e.g., "Lekki Solar Farm")
2. **Location Name** - Specific location name (e.g., "Lekki Peninsula")
3. **Latitude** - Decimal latitude coordinate (e.g., 6.4319)
4. **Longitude** - Decimal longitude coordinate (e.g., 3.5296)
5. **Project Type** - Either "solar" or "cold_chain"
6. **Description** - Project description (optional)
7. **State** - State name (optional)
8. **Local Government** - Local government area (optional)
9. **Installation Date** - Date in YYYY-MM-DD format (optional)

## How to Use:
1. Download this template as Excel file
2. Fill in your installation data
3. Keep the column order exactly as shown
4. Ensure latitude/longitude are in decimal format
5. Use "solar" or "cold_chain" for project type (lowercase)
6. Upload the file using the map interface

## Tips:
- Use decimal format for coordinates (not degrees/minutes/seconds)
- Keep file size under 10MB
- Supported formats: XLSX, XLS, CSV
- All required fields (first 5 columns) must be filled
- Optional fields can be left blank
